# 📱 QR Code Generator in Python

This is a simple Python script that generates a QR code for any link you provide. The generated QR code is saved as an image file with a custom color scheme.

---

## 🎯 Features

- ✅ Generates a QR code from a user-provided link  
- 🎨 Customizable QR appearance (black foreground, yellow background)  
- 🖼️ Saves the QR code as a `.png` image  
- 💡 Easy to use and modify  

---

## 🧑‍💻 Technologies Used

- Python 3.6+
- [`qrcode`](https://pypi.org/project/qrcode/) Python library
